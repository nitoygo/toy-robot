## How to Use

### Pre-requisites

C++ compiler and runtime environment are available.  

### Building and Running the Demo Application

In the `src` directory, run the `make` command:
```
$ cd src
$ make
```

The output executable is in `build/` directory and can be run as follows:
```
$ ./build/ToyRobotDemo
PLACE 1,2,EAST
MOVE
MOVE
LEFT
MOVE
REPORT
Output: 3,3,NORTH
```
